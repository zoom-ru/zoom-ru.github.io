    我们的团队

    我们以关爱为先导（关爱是我们的核心价值观），并且都热衷于沟通世界。了解 Zoom 团队成员。

    管理团队

    Eric S. Yuan

    创始人兼首席执行官

    袁征于 2011 年创立 Zoom，旨在传递幸福，为人们打造无障碍视频协作环境。Zoom 通信平台持续改变全球组织联系、通信和协作的方式。作为公司首席执行官，袁征带领 Zoom 成为 2019 年表现最佳的科技 IPO 之一。

     

    2017 年，Business Insider 将袁征评为“企业科技最具影响力人物”。2018年，Glassdoor 将他评为美国大型公司最佳 CEO。2019 年，他作为全球商业变革引领者登上“彭博 50 人”榜单。2020 年，他入选《时代》周刊“年度商业人物”和“100 位最具影响力人物”榜单。2021 年，他还被 Comparably 评为“倡导多元化的最佳 CEO”。

     

    在创立 Zoom 前，袁征曾担任 Cisco 工程副总裁，负责 Cisco 协作软件的开发。他还是 Webex 的创始工程师之一，并且曾担任 Webex 工程副总裁。

     

    袁征在实时协作领域拥有 11 项专利，另有 20 项发明正在申请专利。

    Aparna Bawa

    首席运营官

    Aparna 为 Zoom 带来了快速成长的技术公司所必需的运营和基础设施建设方面的专业知识。她是我们的首席运营官。在此之前，她曾是我们的首席法务官。就在加入 Zoom 之前，她曾担任 Magento Commerce 的高级副总裁兼总法律顾问，该公司于 2018 年被 Adobe 收购。在入职 Magento 之前，Aparna 曾担任 Nimble Storage (NYSE:NMBL) 的副总裁兼总法律顾问；她在 Nimble 公司 2013 年的首次公开募股中发挥了重要作用。2017 年，她领导完成了 Nimble 向 Hewlett Packard Enterprise 的出售事宜。早前，Aparna 曾领导 Inphi Corp（纽约证券交易所代码：IPHI）的法律和公司发展部门。Bawa 职业生涯的起点是担任 Wilson Sonsini Goodrich & Rosati 的公司和证券律师。她也曾担任 Lehman Brothers and Deutsche Bank 的投资银行家，为技术领域的客户完成了多项首次公开募股和并购交易。

    Velchamy Sankarlingam

    产品和工程总裁

    Velchamy Sankarlingam 是 Zoom 的产品和工程总裁。效力于 Zoom 前，Velchamy 曾是 VMware 的云服务开发与运营高级副总裁，职责范围涵盖研发与 IT。他管理一个研发业务部门，同时也负责公司内的所有基础设施，包括建立和运营最大的私有云之一。Velchamy 还负责帮助 VMware 向 SaaS 转变以及 VMware 所有业务应用程序的开发。加入 VMware 前，Velchamy 曾担任 Cisco 的工程和运营副总裁，负责 Webex 的设计和运营。在此之前，Velchamy 在 Webex（被 Cisco 收购）和 Presenter.com（被 Webex 收购）担任副总裁职位。他还曾效力于 Andersen Consulting（现为 Accenture）、IBM、Network Computing Devices 和 Standard Microsystems。Velchamy 是 Bellus3D 的董事会成员。

    Kelly Steckelberg

    首席财务官

    Kelly Steckelberg 自 2017 年以来一直担任 Zoom Video Communications 的首席财务官，在她的带领下，公司于 2019 年成功上市，并在最近几年实现爆炸式增长。此前，她曾在多家科技公司担任过多个职务（包括 Zoosk 的首席执行官），并在 Cisco、Webex、Epiphany、PeopleSoft 和 KPMG 的财务部门担任过多个职务。Kelly 是 James Avery Artisan Jewelry 的董事会成员。她取得了德克萨斯大学奥斯汀分校会计学硕士学位。

    Graeme Geddes

    首席销售和增长官

    作为 Zoom 的首席销售和增长官，Graeme Geddes 负责在全球范围内领导 Zoom 的销售、解决方案设计、销售运营、赋能和渠道团队。

     

    Graeme 在协作行业有 20 年的丰富经验，于 2019 年加入 Zoom 担任 Zoom 电话业务总经理，从产品发布到售出超过 550 万个许可证仅花了四年不到的时间。他还率先提出了 Zoom 最近许多平台创新的成功进入市场战略，包括 Zoom Contact Center、Zoom Rooms 和面向销售的 Zoom IQ。

     

    在加入 Zoom 之前，Graeme 在 Cisco 担任了近 16 年的领导职务，完成了 Webex 会议、团队消息和视频终端的美洲协作销售总监任期。他还在加州林业和消防部门担任了近四年的消防员职务。

     

    他拥有加州理工州立大学圣路易斯奥比斯波分校工商管理学士学位和斯坦福大学商学院创新领导力证书。

    Michael Adams

    首席信息安全官

    Michael Adams 是 Zoom 的首席信息安全官，拥有近 30 年的安全和领导经验。Michael 于 2020 年 8 月加入 Zoom，担任首席运营官和首席信息安全官的首席法律顾问，同时构建公司内幕风险、全球情报、运营保证和安全法律计划。Michael 毕业于美国海军学院，以工程师身份开始了他的职业生涯，他曾是两位参谋长联席会议主席、许多著名上市公司和私营公司以及美国政府最高层的顾问。他曾在 Palantir 担任高管，并在一家大型国际律师事务所担任合伙人，全都取得了成功。Michael 和他的妻子、两个孩子以及切萨皮克海湾寻回犬生活在北卡罗来纳州夏洛特，在那里他们是社区的积极成员，也是巴尔的摩金莺队的铁杆粉丝。

    Abhisht Arora

    首席战略官

    作为 Zoom 的首席战略官，Abhisht 负责制定关键的发展计划和战略规划。在加入 Zoom 前，他是 Microsoft Teams 的产品副总裁。他在 Microsoft 任职 20 余年，担任过高级业务和产品领导职务，他负责的产品包括 Xbox、Bing、Windows、Office 和 Surface。他在为客户提供产品创新方面有着成功的战绩，例如，创建 Xbox Game Pass，以及推动 Bing 和 Xbox Live 的步进功能使用和收入增长。Abhisht 管理着拥有多元化生态系统的规模平台业务，跨广告、订阅、交易、许可和设备销售等不同业务模式。Abhisht 拥有北卡罗来纳大学教堂山分校凯南- 弗拉格勒商学院的 MBA 学位以及孟买大学商科硕士学位，是一名注册会计师（印度）。

    Wendy Bergh

    线上业务总经理

    Wendy Bergh 是 Zoom 线上业务总经理，通过领导线上市场推广、产品、工程和支付团队，负责公司线上收入增长和客户体验。

     

    Wendy 有 20 多年领导一流消费品牌和企业品牌的经验，在电子商务、营销和移动领域拥有深厚的专业知识。在加入 Zoom 之前，Wendy 曾在 Service Now、Minted 和 Walmart 担任领导职务。Wendy 拥有斯坦福大学商学院的 MBA 学位和科尔比学院的学士学位。

    Shane Crehan

    首席会计官

    Shane Crehan 是 Zoom 的首席会计官，在技术、金融科技和金融服务领域拥有丰富经验。加入 Zoom 前，他曾在 Eventbrite 担任首席会计官。此前他还曾在 Facebook 任职八年，担任过包括国际金融和全球金融业务主管在内的多种职位。Shane Crehan 拥有利默里克大学法律和会计学士学位，同时也是爱尔兰特许会计师协会的注册会计师。Shane Crehan 现居于门洛帕克，喜欢滑雪、徒步和读书。

    Mu Han

    首席开发官

    Mu Han 是 Zoom 的首席开发官，负责领导 Zoom 的产品开发工作。在加入 Zoom 前，Mu 在 Microsoft 工作了 25 年，负责开发通信产品。他曾担任公司副总裁，负责为 Microsoft Teams 和 Skype 提供实时通信技术和服务。他在开发 Teams 呼叫和会议场景的过程中发挥了重要作用，并在疫情期间负责拓展业务。在此之前，他担任过多个领导职务，负责开发各个版本的 Microsoft 统一通信产品。他从 Windows 开发人员做起，并在 Windows Messenger 中实现了视频通话。

     

    Mu 拥有伊利诺伊大学厄巴纳-香槟分校计算机科学硕士学位和清华大学计算机科学学士学位。

    Smita Hashim

    首席产品官

    Smita 是 Zoom 的首席产品官，主要负责 Zoom 的产品管理工作。她曾在 Google 和 Microsoft 任职，是一位拥有 20 多年产品管理经验的业内资深人士。她曾在 Microsoft 负责 IP 网络电视传送业务，负责并扩展 Google Meet，后来还在 Microsoft Teams 负责管理实时通信平台和产品。除了视频业务之外，Smita 还负责过 Google Voice（电话）、Google Calendar 和 Tasks、面向本地广告商的 Google AdWords 产品套件，以及 Google Chromebook 设备产品组合和生态系统。

     

    Smita 在印度长大，就读于印度理工学院坎普尔分校并获得电气工程学士学位。她还获得了普林斯顿大学 EECS 硕士学位和加州大学圣塔芭芭拉分校图像与视频压缩博士学位。

     

    Smita 是印度理工学院坎普尔分校杰出校友奖的获得者，也是印度理工学院坎普尔分校女性校友网络 WIN 的创始人。

    Xuedong Huang

    首席技术官

    黄学东担任首席技术官 (CTO)。在加入 Zoom 之前，他曾在 Microsoft 担任 Azure AI 首席技术官和技术研究员。他在人工智能领域的职业生涯十分耀眼：1993 年，他组建了 Microsoft 的语音技术团队，带领  Microsoft 的人工智能团队在语音识别、机器翻译、自然语言理解和计算机视觉领域取得了多项业界首个达到人类水平的里程碑式成就，他还是电气与电子工程师协会 (IEEE) 和美国计算机协会 (ACM) 会士，并当选为美国国家工程院和美国艺术与科学院院士。

     

    黄学东于 1989 年获得爱丁堡大学电子工程博士学位（获得英国 ORS 和爱丁堡大学奖学金资助），1984 年获得清华大学计算机科学硕士学位，1982 年获得湖南大学计算机科学学士学位。

    Brendan Ittelson

    首席生态系统官

    Brendan Ittelson 是领导 Zoom 开发者生态系统的首席生态系统官。  Brendan 在 2015 年加入 Zoom，担任 Zoom 的全球客户支持主管，并于 2020 年改任首席技术官。效力于 Zoom 之前，Brendan 既是联邦部门的一名承包商，也是国防部的文职雇员，负责创建、实施和支持各种全球身份管理系统。离开国防部后，Brendan 进入金融行业并加入一家名为 Dynamics Inc 的金融科技初创企业，任特别运营副总裁。担任该职务时，他将自己的业务和技术背景相结合，实施了一系列特别项目，包括在不影响支付系统工作方式的情况下革新支付方式。Brendan 拥有俄勒冈大学工商管理专业的学士学位以及海军研究生院身份管理专业文凭。

    Cheree McAlpine

    首席法务官

    Cheree McAlpine 是 Zoom 的首席法务官，她拥有超过 25 年代表全球科技公司水平的法务经验。 

     

    Cheree 曾担任过多个高级法务职位，最近担任的职位是 Intercom（一家为客户提供解决方案的人工智能公司）的高级副总裁兼法律总顾问。在此之前，她是 Lumileds Inc.  的首席法务官兼公司秘书。在 Lumileds 任职之前，她曾担任 Avnet Americas 和 Wyse Technology 的法律总顾问，成功地将公司出售给 Dell。

     

    Cheree 拥有阿拉巴马大学学士学位和萨福克大学法学院法律学位。她现居湾区。

    Matthew Saxon

    首席人力官
